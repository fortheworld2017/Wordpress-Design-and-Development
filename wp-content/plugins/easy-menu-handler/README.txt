Easy Menu Handler (EMH) provides you with a fast and user-friendly way to delete a single menu item or it's whole branch. 

== Description ==

Time is money and being able to complete your tasks faster is what matters.

With Easy Menu Handler (EMH) two new buttons will be added to your Wordpress Menu items
which will provide you with the functionality to delete a specific menu item or menu branch
directly, without the need to first open the menu tab. 

This is extremely helpful especially to developers/users who have just installed a brand new 
theme and imported the demo content which will populate their menu's with very long menu structures.

== Installation ==

1. Upload EasyMenuHandler to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress

Alternative,

1. Navigate to Plugins > Add New
2. Select 'Upload Plugin'
3. Click on 'Browse...' and select the EasyMenuHandler zip file and click 'Install Now'
4. Activate the plugin through the ‘Plugins’ menu in WordPress

After that two new buttons should appear on your menu items toolbar's under Appearance > Menus.
